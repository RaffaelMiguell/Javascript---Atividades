# OPERADORES JS

## ATRIBUIÇÃO

| OPERADORES |   EXEMPLO   | EQUIVALENTE |
| :--------: | :---------: | :---------: |
|     =      |     X=Y     |     X=Y     |
|  += / -=   | X+=Y / X-=Y |    X=X+Y    |
|    \* =    |    X\*=Y    |   X=X\*Y    |
|     /=     |    X/=Y     |    X=X/Y    |
|     %=     |    X%=Y     |    X=X%Y    |

## COMPARAÇÃO

| OPERADORES |       EXEMPLO        |
| :--------: | :------------------: |
|     ==     |        IGUAL         |
|    ===     | COMPARA VALOR E TIPO |
|     !=     |      DIFERENTE       |
|    !==     | VALOR E TIPO DIFER.  |
|     >      |        MAIOR         |
|     <      |        MENOR         |
|     >=     |     MAIOR IGUAL      |
|     <=     |     MENOR IGUAL      |
